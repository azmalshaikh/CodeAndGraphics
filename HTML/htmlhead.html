HTML Head

Prev < Next >

Container for many elements such as meta, title, link etc.

We will Discuss these - <title>, <style>, <meta>, <link>, <script> and <base> .

Metadata (data about data) define the document title, character set, styles, scripts, link, and other meta information



1. <title> :

  • Defines title in browser.
  • This tag is required in HTML documents
  • Important for SEO (Search engine optimization)

Example

<head>
  <title> Webpage </title>
</head>

2. <style> :

  • To specify style information in HTML page
  • In other words To add CSS in html file
  • Syntax is same as in CSS file

Example

<head>
  <style> //CSS Code</style>
</head>

3. <link> :

  • Used to link an external file to the current file
  • Usually used to link external style sheet (CSS file)

Example

<head>
  <link rel="stylesheet" href="mystyle.css">
</head>

4. <meta>:

  • Specify the Description of webpage, keywords, author, Character set, Viewport etc
  • These are used by browser, and search engines.

Character Set

<meta charset="UTF-8">

Note: UTF-8 covers almost al of character and symbols.

Description of webpage

<meta name="description" content="Web, Programming and Graphic design Tutorials">

Keywords for Search Engines

<meta name="keywords" content="HTML, CSS, JS, Photoshop, Illustrator, PremierePro, Java, C++, Python">

Author of Webpage

<meta name="author" content="Name">

Setting the viewport

<meta name="viewport" content="width=device-width, initial scale=1.0">

Viewport is user's visible area of webpage.
The width=device-width part sets the width of page to follow the screen-width of device.
The initial-scale = 1.0 part sets the initial zoom level when the page is first loaded by browser.

5. <script>:

  • To define Javascripts in HTML file

Example

<head>
  <script> //JS Code</script>
</head>
