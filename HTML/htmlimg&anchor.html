HTML Img & Anchor

Prev < Next >

1. Image :

  • Denotations : <img>
  • Location : Inside Body tag
  • Usage : To add an image to the webpage
  • Syntax : <img src="url">
    Here src is attribute to incluse image's path/url. Image mainly contains attributes namely alt , width and height.

Example :

<img src="url">


The alt Attribute

  • Defines text/name for an image (In case it can't be displayed).
  • Value of alt attribute should described about image.

    • Example :

    <img src="rose.jpg" alt="Rose Flower">

    Note : The alt attribute is required. A web page will not validate correctly without it.



    The width and height

  • Attribute to set the width and height of image.
  • The width and height attributes always set the width and height of the image in pixels.

    • Example :

    <img src="rose.jpg" width="300" height="400">

    Note : Always specify the width and height of an image.


    Tip : Now pracitce image tag and it's attribute (alt, width, height)
