HTML List & Tables

Prev < Next >

HTML List :

  • Usage : To group a set of related items in lists

Example :

  • List 1
  • List 2
  • List 3
  • List 4

1. Unordered list :

  • Collection of related items that have no special order or sequence
  • Starts with <ul> tag and this will list items using plain bullets
  • Each list item starts with the <li> tag

    • Example :

    <ul>
      <li> C </li>
      <li> C++ </li>
      <li> Java </li>
      <li> Python </li>
    </ul>

    Preview :

    • C
    • C++
    • Java
    • Python

    Unordered list Type Attribute :

  • The type attribute of the <ul> tag, defines the type of sequence.

    • Type Attribute Discription/Meaning
    1 type="disc" Sets the list item marker to a bullet (default)
    2 type="square" Sets the list item marker to a square
    3 type="circle" Sets the list item marker to a circle

    2. Ordered list :

  • Collection of related items that have special order or sequence
  • Each list item starts with the <li> tag
  • Items will be marked with numbers by default

    • Example :

    <ol>
      <li> C </li>
      <li> C++ </li>
      <li> Java </li>
      <li> Python </li>
    </ol>

    Preview :

      1. C
      2. C++
      3. Java
      4. Python

    Ordered list Type Attribute :

  • The type attribute of the <ol> tag, defines the type of sequence.

    • Type Attribute Discription/Meaning
    1 type="1" List items will be sequenced with numbers(default)
    2 type="i" List items will be sequenced with lowercase roman numbers
    3 type="I" List items will be sequenced with uppercase roman numbers
    4 type="A" List items will be sequenced with uppercase letters
    5 type="a" List items will be sequenced with lowercase letters

    Note : You can also use the start attribute in HTML ordered list
    <ol type="i" start="4">


    3. Defination list :

  • Entries are listed like in a dictionary
  • Usage : To present a glossary , list of items or any other value list
  • Tags:
      <dl> - Defines the start of list.
      <dt> - A term
      <dd> - Term Defination
      </dl> - Defines the end of list.

    • Example :

    <dl>
      <dt> A term (name)</dt>
      <dd> Term Defination</dd>
      <dt> Java Script</dt>
      <dd> Programming language used to make interactive webpages</dd>
    </dl>

    Preview :

      A term
      Term defination
      Java Script
      Programming language used to make interactive webpages

    Note : In HTML list nesting is allowed. It means list inside lists. Explore this concept yourself!!


    HTML Tables :

    • An HTML Table is defined with <table> tag
    • Each table row is defined with <tr>tag
    • A table header is defined with <th> tag (By default table heading are bold and centered)
    • A table data cell is defined with <td> tag

    Example :

    <table>
      <tr>
       <th> Name</th>
       <th> Age</th>
       <th> Qualification </th>
      </tr>
      <tr>
       <td> Rohan</td>
       <td> 26</td>
       <td> Android Developer </td>
      </tr>
      <tr>
       <td> Harry</td>
       <td> 20</td>
       <td> Web & Android developer </td>
      </tr>
    </table>

    Preview :

                   
      Name Age Qualification
      Rohan 26 Android developer
      Harry 20 Web & Android developer

    Note : For adding a border for the table CSS border property is used. We will discuss this in CSS tables section. In CSS tables we will explore about colspan and rowspan attribute , Caption for table and styles for table.
