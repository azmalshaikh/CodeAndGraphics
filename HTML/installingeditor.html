Installing Editor

Prev < Next >

Different types of Editor :

  1. Simple Text Editor :
    • Notepad (Windows)
    • TextEdit (Mac)
      • Note : The above two are simple text editor which doesn't have advanced features.
  2. Advanced Text Editor :
    • Brackets = Bracket is an open-source software primarily used for Web development. It provides live HTML, CSS, JavaScript editing functionality.
    • VisualStudio Code = Visual Studio Code is a code editor redefined and optimized for building and debugging modern web and cloud applications.

Brackets:

A modern, open source text editor that understands web design.

    Main Features :
  • Inline Editors
  • Live Preview
  • Preprocessor Support
    • Popular Extensions :
  • Emmet
  • Beautify
  • File Icon
  • Indent Guides
  • Git
  • Auto Save

VisualStudio Code:

A modern, open source text editor that understands web design.

    Main Features :
  • IntelliSense
  • Live Preview
  • Run and Debug
  • Built in Git
    • Popular Extensions :
    Want even more features? Install extensions to add new languages, themes, debuggers, and to connect to additional services. Extensions run in separate processes, ensuring they won't slow down your editor.
