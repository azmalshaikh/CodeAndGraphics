HTML Entities

Prev < Next >

Some character are reserved in HTML so they must be replaced with character entities.

For example : If you use < or > sign the browser might mix them with tags.

Character entities are used to display reserved characters in HTML.

Syntax : &entity_name; OR &#entity_number;

Generally, entity name is used because name is easy to rememeber and support all type of browser's.



Some Useful HTML Character Entities

Character Name Entity Name Entity Number
< less than &lt; &#60;
> greator than &gt; &#62;
& ampersand &amp; &#38;
" double quotation mark &quot; &#34;
' single quotation mark (apostrophe) &apos; &#39;
£ pound &pound; &#163;
€ euro &euro; &#8364;
© copyright &copy; &#169;
® Registered trademark &reg; &#174;
Non breaking space &nbsp; &#160;
∀ For All &forall; &#8704;
∂ Partial Differential &part; &#8706;
∃ There Exists &exist; &#8707;
∅ Empty Sets &empty; &#8711;
∑ Summation &sum; &#8721;
∏ N-ary Product &prod; &#8719;
∈ Element of &isin; &#8721;
Α GREEK CAPITAL LETTER ALPHA &Alpha; &#913;
Β GREEK CAPITAL LETTER BETA &Beta; &#914;

Note : Entity names are case sensitive.
