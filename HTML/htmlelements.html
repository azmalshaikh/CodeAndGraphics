HTML Elements

Prev < Next >

Defined by a start tag, some content, and end tag.

  <tagname> Content </tagname>

Examples :
  <h1> Heading </h1>
  <p> Paragraph </p>
  <a> Hyperlinks </a>

Start tag Content End tag
1 <h1> Heading </h1>
2 <p> Paragraph </p>
3 <a> Hyperlinks </a>

Note : Never skip the end tag. However some HTML elements will display correctly, even if you forget the end tag But Unexpected results and errors may occur if you forget the end tag!


Note : HTML tags are not case-sensitive. So <P> and <p> are same.
However it is recommended to use lowercase in whole HTML document. At CodeAndGraphics we always use lowercase tag names.


Note : In HTML nesting is allowed. Nesting means putting one tag inside another tag.
Example :
  <body>
    <p>
    This is the paragraph.
    </p>
  </body>
In this example the <p> is inside the <body>
