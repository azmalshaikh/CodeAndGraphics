HTML Headings and paragraphs

Prev < Next >

1. Headings :

  • Denotations : <h1></h1> , <h2></h2> , <h3></h3> ......
  • Location : Inside the body tag
  • Usage : Title/Heading you have to give to a paragraph or any other section
  • <h1> The most important heading (font-size = larger)
  • h1 followed by h2 headings and so on (Decreasing order of font-size)

Example :

<!DOCTYPE html>
<html>
<head>
  <title> Headings</title>
</head>
<body>
  <h1> Heading 1 <h1>
  <h2> Heading 2 <h2>
  <h3> Heading 3 <h3>
  <h4> Heading 4 <h4>

</body>
</html>

Preview :

Heading 1
Heading 2
Heading 3
Heading 4

Note : Use HTML headings for headings only. Don't use headings to make text BIG or bold.


2. Paragraph :

  • Denotation : <p></p>
  • Location : Inside the body tag
  • Usage : Make new paragraph
  • Visibility : Text placed inside this tag will visible on main webpage.
  • A Paragraph always start on a new line

Example :

<!DOCTYPE html>
<html>
<head>
  <title> Headings</title>
</head>
<body>
  <p> This is paragraph. Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repellat, beatae. <p>

</body>
</html>

Preview :

This is paragraph. Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repellat, beatae.

Note : In HTML you cannot change the display by adding extra spaces in your code. The browser will automatically remove any extra spaces when page is displayed.


    Break tag :
  • Denotations : <br>
  • Location : Inside the body tag
  • Usage : Lets you start from a new line
  • Special characteristics : It's a self closing HTML tag. (No closing tag)
