CSS Borders

Prev < Next >

The border properties allow you to specify how the border of the box representing an element should look.

  • Border Width
  • Border Style
  • Border Color

    • 1. Border Width :

    • Used to specify width of the border.
    • border-width: 10px;
    • It can take values in px, thick, medium or thin.

    2. Border Style :

    • Used to specify style of the border.
    • border-style: solid;
    • It can take values as solid, dotted, dashed, double, groove, none ...

    3. Border Color :

    • Used to specify color of the border.
    • border-color: black;

    Note : Shorthand = border: width style color;
